●機能とか譜面の命令文とか

◎操作方法

keyconfig.iniに書き込んだキーデータで操作が可能になります。
使っているライブラリの使用上、キーデータの指定方法が太鼓さん次郎などとは異なります。

デフォルトではS、D、K、Lが縁でF、G、H、Jが面です。

スラッシュキーを押すとコマンド入力モードになりコマンド一覧が出てきます
コマンドの効果は自分で確かめてみよう！(=めんどい)

config.iniにてTJACreateModeを1にすると譜面編集に特化したモードになります
その時の操作方法を一応書いておきます。

Qキー -> 譜面再読み込み
PgUp -> 一小節進める
PgDn -> 一小節戻る
Home -> 譜面の先頭に行く
End  -> 譜面の末尾に行く

◎対応してる譜面の命令文(新規命令文もあります)

#START
■譜面データの開始を宣言します、当然これがないと譜面は流れません。

#END
■譜面データの終了を宣言します、コース分けをする際は必須です。

#SCROLL <rate>
■譜面の流れる速度をn倍します。

#SCROLL <Realrate>+<Imaginaryrate>i
■譜面の流れる速度をa+bi倍します(例:#SCROLL 1+2i)。

#SCROLL <radius>,<divvalue>,<angle>
■極座標r,θからなるスクロール効果です、rには距離、dには円の分割数、θには角度を入力すると対応した角度、距離から流すことができます。

#BPMCHANGE <beattempo>
■譜面のBPMをnに設定します。

#MEASURE <Left>/<Right>
■小節の長さを分数で表した長さに変えます。

#GOGOSTART
■ゴーゴータイムを開始します。

#GOGOEND
■ゴーゴータイムを終了します。

#BARLINEON
■小節線を表示します。

#BARLINOFF
■小節線を消します。

#DELAY <second>
■指定した秒数分譜面をずらします。

#BRANCHSTART <type>,<Expart>,<Mastar>
■譜面分岐の種類を指定してから(pで精度分岐、rで連打分岐、sでスコア分岐となる)、玄人分岐させるための数値を指定し、達人分岐させるための数値を指定(N < E < M)。

#N
■これを書いたところから普通譜面を書く。

#E
■これを書いたところから玄人譜面を書く。

#M
■これを書いたところから達人譜面を書く。

#BRANCHEND
■譜面分岐を正常終了させるためのもの。

#JUDGEDELAY <x> <y> <z> <w>
■xが0の時通常の位置に戻す、xが1か3の時yに秒数を入力または2の場合x軸の座標入力をする、xが2の時zにy軸の座標入力または3の場合x軸の座標入力をする、xが3の時y軸の座標入力をする。

#DUMMYSTART
■ダミーノーツになります。

#DUMMYEND
■普通のノーツになります。

#NOTESPAWN <type> <second>
■typeが0の時その場所から何も起こらない状態にできる、typeが1の時出現する位置を秒数で指定、typeが2の時透明化させる位置を秒数で指定。

#LYRIC <string>
■stringに書いた歌詞を表示させます。

#SUDDEN <second> <second>
■a秒前に出現して、b秒前に動き出します。

#JPOSSCROLL <second> <motionpx> <type>
■移動時間、移動距離、方向を指定、移動距離の指定については別txtファイルで解説してます

#SIZE <rate>
■ノーツのサイズをx倍します

#COLOR <red> <green> <blue> <alpha>
■ノーツの色彩を変えられます。

#ANGLE <angle>
■ノーツの画像をn度回転させます

#GRADATION <type> <second> <type1> <type2>
■説明むずいので書き方を解説しているtxtファイルを見て下さい

#BARLINESIZE <width> <height>
■小節線の長さや幅を変更できます

#RESETCOMMAND
■すべての命令文の効果を初期値に戻します
