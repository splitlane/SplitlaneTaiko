GRADATIONSTARTとGRADATIONENDの説明が書いてあるファイルです、
ギミック譜面を作る方はどうぞ参考にしてみてください。

まず基本的な書き方を書いておきます

#GRADATION start <second> <type1> <type2>
対応している命令文で変化前の値を記入
#GRADATION end
対応している命令文で変化後の値を記入
譜面,
#GRADATION init
//必要に応じて初期化

こんなかんじです、
ぱっと見よくわかりませんね()
では解説していきます。

とりあえずまずは秒数を入れましょう()

そしてtype1とtype2ってなんぞという話なんですが、
これらはイージング関数の挙動を使うために実装しました、
type1には「In」と「Out」と「InOut」の三種類あるうちのどれかを数字で記入します(0、1、2)。

type2には挙動のタイプを数字で記入します、
対応してるのは以下の11種類です。

Linear  = 直線　　　　(0)	
Sine    = 三角関数　　(1)
Quad    = 二次　　　　(2)
Cubic   = 三次　　　　(3)
Quart   = 四次　　　　(4)
Quint   = 五次　　　　(5)
Expo    = 指数　　   (6)
Circ    = 円形　　   (7)
Back    = 三次と二次　(8)
Elastic = 弾性　　   (9)
Bounce  = バウンド   (10)

そしたらあとは対応している命令文を書いたりするだけです、
GRADATIONに対応している命令文は以下の5種類です。

#SCROLL
#JUDGEDELAY
#SIZE
#COLOR
#ANGLE

最後に、この命令文便利な代わりに超重いです、
処理的にかなりの計算量があるのでほかの命令文に比べたら雲泥の差です。
なので頻繁に使うと譜面が重くなることは承知の上でお願いします。
